Thai Consonant Drill Application Guide

System Overview

The Thai Consonant Drill is a full-stack web application designed to help users master Thai consonants through an interactive quiz. It features persistent play count tracking using a PostgreSQL database.

Backend API

POST /start-quiz

Initializes or updates a player's session and increments their total play count.

Request Body:

{
  "nickname": "String"
}

Response:

{
  "streak_count": Number
}

*Note: Though named streak_count in the database for legacy reasons, it now represents total plays.

GET /health

Returns the status of the server.

Deployment on Render

  1. Database: Create a New PostgreSQL instance. Use the Internal Database URL for the web service.
  2. Web Service: Create a New Web Service connected to your GitHub repo.
    • Root Directory: backend
    • Build Command: npm install
    • Start Command: node server.js
  3. Environment Variables: Add DATABASE_URL with your PostgreSQL connection string.
Tip: Ensure your Render service has access to the database via its internal network for faster performance.

Accessing Your Database

You can access your PostgreSQL data in two main ways:

1. Render Dashboard (Visual)

  • Log in to Render.
  • Select your PostgreSQL instance.
  • Go to the Connect tab to see connection details.
  • Use the Shell tab to run SQL queries directly in the browser.

2. External Management Tools

To use tools like DBeaver, TablePlus, or psql:

  • Copy the External Database URL from Render.
  • Paste it into your tool's "Connect with URL" or "Import URI" section.
Important: Render databases may require you to add your IP address to the Access Control settings before you can connect externally.
